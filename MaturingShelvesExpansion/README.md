# Maturing Shelves Expansion

Adds new recipes to the Maturing Shelves machine from [Visit Mount Vapius](https://www.nexusmods.com/stardewvalley/mods/9600). Grow mushrooms, multiply foraged items, and convert crops into useful ingredients. (I had too many maturing shelves and not enough cheese/milk.)

## Recipes

All recipes also consume Hay x3 (built into the machine by VMV).

### Always Available (requires Visit Mount Vapius)

| Input | Output | Time |
|---|---|---|
| Hardwood x1 | Woodgoat Mushroom x1-3 | 2 days |
| Spring Onion x1 | Spring Onion x2-4 | 3 days |
| Coffee Bean x3 | Morel x1 | 2 days |
| Wood x10 | Porchini x1-3 | 2 days |
| Wheat x5 | Common Mushroom x1-3 | 1 day |
| Basil x5 (VMV) | Tea Leaves x5 | 1 day |

### Ridgeside Village (optional)

These recipes only appear if [Ridgeside Village](https://www.nexusmods.com/stardewvalley/mods/7286) is installed.

| Input | Output | Time |
|---|---|---|
| Mountain Arugula x1 | Mountain Arugula x2-4 | 3 days |

## Requirements

- [SMAPI](https://smapi.io/)
- [Content Patcher](https://www.nexusmods.com/stardewvalley/mods/1915) 2.0+
- [Visit Mount Vapius](https://www.nexusmods.com/stardewvalley/mods/9600) (required)
- [Ridgeside Village](https://www.nexusmods.com/stardewvalley/mods/7286) (optional, for Mountain Arugula recipe)

## Install

1. Install the required mods listed above
2. Drop the `MaturingShelvesExpansion` folder into your `Mods` directory
3. Run the game
