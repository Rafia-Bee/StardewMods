# Stardew Mods

A collection of my mods for [Stardew Valley](https://www.stardewvalley.net/).

## Mods

| Mod | Description |
| --- | --- |
| [Bigger Auto-Grabber](BiggerAutoGrabber/) | Expands the auto-grabber's internal chest beyond the vanilla 36-slot limit |
| [Bulk Derby Rewards](BulkDerbyRewards/) | Collect all your Trout Derby rewards at once instead of one at a time |
| [Catch of The Day](CatchOfTheDay/) | Mod that shows weather or time exclusive fish icons on the HUD |
| [Deluxe Grabber Fix](DeluxeGrabberFix/) | Fixed and improved fork of Deluxe Grabber Redux for SDV 1.6 |
| [Livestock Follows You](LivestockFollowsYou/) | When you buy an animal, it follows you around instead of teleporting straight to its building |
| [Loanable Tractor](LoanableTractor/) | Rent a tractor from Joja Corp. for a daily fee — no garage required |
